# **Steps to Use FTK Imager**

**FTK Imager** is a digital forensic tool used to **collect, preview, and preserve digital evidence** without modifying the original data.

---

## **Procedure**

### **Step 1: Install FTK Imager**

1. Download **FTK Imager** from the official website.
2. Install the software on your computer.
3. Launch the application after installation.

---

### **Step 2: Create a Disk Image**

1. Open **FTK Imager**.
2. Click **File** → **Create Disk Image**.
3. Select the evidence source:

   * Physical Drive
   * Logical Drive
   * Image File
   * Folder Contents
4. Click **Next**.

---

### **Step 3: Select Evidence Source**

1. Choose the drive or folder that contains the data.
2. Select the required disk or partition.
3. Click **Finish**.

---

### **Step 4: Choose Image Type**

1. Select the image format:

   * **RAW (dd)** – Standard forensic image format
   * **E01 (Expert Witness Format)** – Most commonly used
   * **SMART**
   * **AFF**
2. Click **Next**.

---

### **Step 5: Enter Case Information**

Fill in the required details such as:

* Case Number
* Evidence Number
* Examiner Name
* Description

Click **Next** after entering the details.

---

### **Step 6: Select Destination Folder**

1. Click **Add** to choose where the image file will be saved.
2. Select destination folder.
3. Enter file name.
4. Click **Next**.

---

### **Step 7: Create Image**

1. Click **Start** to begin the imaging process.
2. The software will create a forensic image of the selected data.
3. Wait until the process is completed.

---

### **Step 8: Verify Image Integrity**

1. Enable **Verify Images After Creation** option.
2. The tool generates **hash values (MD5/SHA1)**.
3. These values ensure that the copied data matches the original data.

---

### **Step 9: Preview Evidence**

1. Click **File** → **Add Evidence Item**.
2. Select the created image file.
3. Browse files, folders, and contents.
4. Capture screenshots or export files if needed.

---

# **Result**

Digital evidence was successfully collected and preserved using **FTK Imager**, and the integrity of the data was verified using hash values.

---

# **Conclusion (Short – if needed)**

FTK Imager is an effective forensic tool used to acquire and preserve digital evidence without altering the original data. It ensures data integrity through hash verification and supports secure evidence collection for cybercrime investigations.
