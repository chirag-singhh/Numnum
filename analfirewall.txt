Aim:
To configure and analyze firewall rules on both Linux and Windows operating systems to
enhance network security.
Tools Required:
1. Computer or virtual machine running Linux (Ubuntu/Kali).
2. Computer or virtual machine running Windows 10/11.
3. UFW (Uncomplicated Firewall) for Linux.
4. Windows Defender Firewall with Advanced Security for Windows.
Theory:
A firewall is a security system that monitors and controls incoming and outgoing network
traffic based on predefined security rules. Firewalls help prevent unauthorized access,
mitigate attacks, and enforce network policies.
Types of Firewalls:
 Packet Filtering Firewall – Analyzes network packets based on IP, port, and
protocol.
 Stateful Inspection Firewall – Tracks the state of active connections.
 Application Layer Firewall – Inspects traffic at the application level.
Firewall Tools Used:
 Linux: Uses UFW (Uncomplicated Firewall) to manage iptables rules easily.
 Windows: Uses Windows Defender Firewall to define inbound/outbound traffic
rules.v

Step 2: Configure Firewall on Windows
1. Open Windows Defender Firewall with Advanced Security.
2. Navigate to Inbound Rules and Outbound Rules.
3. Create a new rule to allow traffic on ports 22 (SSH), 80 (HTTP), and 443 (HTTPS).
4. Block all traffic except the allowed rules using the default deny policy.
5. Verify the rules by checking firewall logs.
