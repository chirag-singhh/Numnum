How to create Rules in the firewall-

Step 1: Open Windows Defender Firewall with Advanced Security
1. Press Win + R, type wf.msc, and press Enter.
2. This opens the Windows Defender Firewall with Advanced Security console.

Step 2: Create an Inbound Rule to Block Unauthorized Traffic
Inbound rules control traffic coming into your computer.
1. Navigate to Inbound Rules
 In the left pane, click Inbound Rules.
2. Start Creating a New Rule
 In the right pane, click New Rule.
3. Choose Rule Type
 In the New Inbound Rule Wizard, select Program or Port or Custom, depending on your
need:
o Program – To block traffic for a specific application.
o Port – To block traffic for a specific port (e.g., 80 for HTTP, 443 for HTTPS).
o Custom – For advanced rule configurations.

4. Specify the Program or Port
 If Program:
o Select This program path and browse to the executable file (.exe) of the application
you want to block.

 If Port:
o Select TCP or UDP, and specify the port number(s) to block.

5. Block the Connection
 Select Block the connection, then click Next.
6. Apply the Rule
 Choose when the rule applies:
o Domain – Applies when connected to a domain network.
o Private – Applies to private (home/work) networks.
o Public – Applies to public (Wi-Fi, coffee shops, etc.).

 Click Next.
7. Name and Save the Rule
 Give the rule a descriptive name (e.g., Block Unauthorized HTTP).
 Click Finish.

Step 3: Create an Outbound Rule to Block Unauthorized Traffic
Outbound rules control traffic going out of your computer.
1. Navigate to Outbound Rules
 In the left pane, click Outbound Rules.
2. Start Creating a New Rule
 In the right pane, click New Rule.
3. Choose Rule Type
 Select Program, Port, or Custom, similar to Step 2.
4. Specify the Program or Port
 If Program:
o Select This program path and browse to the executable file (.exe) of the application
you want to block.

 If Port:
o Select TCP or UDP, and specify the port number(s) to block.

5. Block the Connection
 Select Block the connection, then click Next.
6. Apply the Rule
 Choose when the rule applies (Domain, Private, Public).
 Click Next.
7. Name and Save the Rule
 Give the rule a descriptive name (e.g., Block Unauthorized App).
 Click Finish.

Step 4: Verify and Test the Rules
1. Go back to Windows Defender Firewall with Advanced Security.
2. Check that your new rules appear under Inbound Rules and Outbound Rules.

3. Test by trying to access the blocked application or port.

Step 5: Modify or Delete a Rule (if needed)
1. Open Windows Defender Firewall with Advanced Security.
2. Click Inbound Rules or Outbound Rules.
3. Find the rule you created.
4. Right-click the rule and choose Disable, Modify, or Delete.

Additional Security Considerations
 Use Allow Rules Cautiously: By default, Windows Firewall blocks most unauthorized traffic.
Only create Allow rules for trusted apps.
 Enable Logging:
o Open Windows Firewall, go to Monitoring &gt; Logging, and enable it to track blocked
attempts.

 Use Group Policy (GPO) for Network-wide Control (for enterprise environments).
